# SEMCO Smart RFQ - Frontend Web Portal

This repository contains the client-side single page application (SPA) for the SEMCO Smart RFQ Sourcing Portal.

## Technical Details
- **Core Stack:** HTML5, CSS3, Vanilla JavaScript, Bootstrap 5.
- **Features:** Responsive design, live ranking dashboard, vendor portals, transporter portals, multi-screen compatibility down to 320px.

## Running Locally
Since it is a single-page HTML application, you can open `index.html` directly in your web browser or run it with any static web server (such as Live Server in VS Code).
